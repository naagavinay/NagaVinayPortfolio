CUSTOMER LIFETIME VALUE PREDICTION

Developed a predictive model in this project centered on Customer Lifetime Value (CLV) prediction using Python. Performed Exploratory Data Analysis with a strong reliance on key libraries such as Pandas, NumPy, Matplotlib, Plotly, and Lifetime. This project involved the development of a sophisticated framework that forecasts the probability of customers making purchases in the near future, predicts customer RFM (Recency, Frequency, Monetary) scores, and anticipates transaction values for upcoming purchases. By combining data analysis and predictive modeling, I aimed to equip produce actionable insights into customer behavior, facilitating targeted strategies for customer engagement and revenue maximization.
